# 📊 Customer Churn Prediction

This project uses Machine Learning to predict whether a customer will leave a service (churn) or not. It helps businesses take proactive actions to retain customers.

---

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Classification)
- Prediction of Customer Churn

---

## 🧠 Technologies Used
- Python
- Pandas (Data Handling)
- NumPy (Numerical Computation)
- Matplotlib / Seaborn (Data Visualization)
- Scikit-learn (Machine Learning Models)

---

## 📂 Project Structure
- `Untitled1.ipynb` → Main notebook (code + output)
